# Tipografía del Logo - ShopFlow POS

## Fuente Seleccionada

**Inter** - Fuente sans-serif moderna y profesional

**Razones de selección**:
- Diseñada específicamente para interfaces digitales
- Excelente legibilidad en todos los tamaños
- Amplia familia de pesos disponibles
- Optimizada para pantallas
- Ya utilizada en el proyecto (Next.js default)

## Especificaciones de Tipografía

### "ShopFlow"

- **Fuente**: Inter
- **Peso**: 700 (Bold)
- **Tamaño base**: 32px (para logo horizontal de 800x200)
- **Tracking**: -0.02em (ligero ajuste para mejor legibilidad)
- **Altura de línea**: 1.2

### "POS"

- **Fuente**: Inter
- **Peso**: 400 (Regular)
- **Tamaño**: 20px (62.5% del tamaño de "ShopFlow")
- **Tracking**: 0.05em (ligero espaciado para diferenciación)
- **Altura de línea**: 1.2

## Espaciado

- **Espacio entre "ShopFlow" y "POS"**: 8px
- **Espacio entre símbolo y texto**: 16px (mínimo)

## Variaciones de Tamaño

### Logo Horizontal (800x200)
- ShopFlow: 32px
- POS: 20px

### Logo Horizontal @2x (1600x400)
- ShopFlow: 64px
- POS: 40px

### Logo Vertical (300x400)
- ShopFlow: 28px
- POS: 18px

### Logo Vertical @2x (600x800)
- ShopFlow: 56px
- POS: 36px

### Isologo (600x150)
- ShopFlow: 24px
- POS: 15px

### Isologo @2x (1200x300)
- ShopFlow: 48px
- POS: 30px

## Uso en Código

```css
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
font-weight: 700; /* Para ShopFlow */
font-weight: 400; /* Para POS */
```

## Notas

- La fuente Inter está disponible por defecto en Next.js
- Para uso en diseño gráfico, usar la fuente Inter de Google Fonts
- Mantener consistencia en el tracking y spacing en todas las versiones
